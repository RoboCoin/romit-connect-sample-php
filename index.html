Get User

This call requires user authentication with OAuth. When you submit this form, the server will first get a client token, then use that token to begin the authorization process. The first step will be to send an SMS message to the supplied phone number.

When the user replies to the SMS message, Romit's servers will notify the supplied redirect URI with a code. When that happens, the merchant's servers can call Romit's servers to exchange that code for an authorization token.

Once the merchant's servers have the authorization token, they can then make authorized calls like this call to get a user's details.
